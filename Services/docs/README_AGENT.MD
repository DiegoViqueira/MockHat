### Environment Setup

1. **Set up environment**

   ```bash
   conda create -n mockhat-agent python=3.9
   conda activate mockhat-agent
   ```

2. **Install Dependencies**

   ```bash
   pip install -r requirements.txt
   pip install -r requirements.dev.txt
   ```

3. **Run Agent**
   ```bash
   python -m app.agent
   ```
